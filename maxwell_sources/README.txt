James Clerk Maxwell — Sources — Directory Structure

./maxwell_sources/
  /primary_pdfs/            # Archived PDFs (Treatise editions, papers, correspondence)
  /images_figures/          # Extracted figures/equations with captions and page refs
  /manifests/               # Acquisition manifests (URL, date, checksum, notes)
  /extractions/             # Structured extractions (equations, definitions, parameter forms)
  /notes/                   # Exhaustive reading notes with citations
  /links/                   # Plaintext link lists to source locations

Intake checklist
- Save PDF to primary_pdfs/
- Record manifest: URL, access date, SHA256, edition
- Create extraction stub for equations (div E, curl B, energy density, Poynting, stress)
- Snapshot key figures/equations to images_figures/
- Write notes with section/page citations to notes/
